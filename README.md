# 梅州会计云招聘网站

一个专业的财务人才招聘平台，提供会计、财务相关职位的发布和搜索服务。

## 功能特点

- 企业可以发布会计和财务相关职位
- 求职者可以搜索和申请职位
- 职位筛选和排序功能
- 响应式设计，支持移动端访问
- 实时统计数据展示

## 技术栈

- HTML5
- Tailwind CSS
- 原生JavaScript
